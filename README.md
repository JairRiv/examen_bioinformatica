# Examen Bioinformatica y Estadistica 2
**Por: Jair Emiliano Contreras Rivera**

Este repositorio fue creado para demostrar como resolver un ejericio usando las beunas practicas de programacion

El programa busca a partir de un archivo FASTA y un GFF los genes y recupera la secuencia

El ejemplo proporcionado es del virus del VIH, el cual tiene 9 genes por lo que es simple e ideal
